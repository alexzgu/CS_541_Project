We are using the pipenv virtual environment to manage the dependencies.

- `cd` to `CS_541_Project`
- `pipenv install`

In case you ever want to start from a clean slate, you can run the below commands:

- `pipenv --clear`
- `pipenv --rm`
- `pipenv --python 3.10`
- `pipenv install`

Dependencies:
- besides those listed in the pipfile, it's up to you to install `ffmpeg`
